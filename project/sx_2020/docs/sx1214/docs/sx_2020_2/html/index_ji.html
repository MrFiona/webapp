代表你的生肖与今年太岁相害，意味着你今年的人际运整体不佳，容易遭到小人破害，从而对各方面运势都易产生不好的影响。

进入2019年猪年，属鸡人的运势较上一年的害太岁年份而言，各方面都得到了提升。

但今年属鸡人的运势整体来说并不算太好，虽说受吉星的影响，今年属鸡人正财运很旺，在职场中或身处国家公职的人员的属鸡人，今年升职加薪的希望非常大。但是工作中还是要低调谦逊，平常也不要跟同事一起说公司的八卦或领导的坏话，害人之心不可有防人之心不可无，今年属鸡人容易受到小人的困扰，所以一切还是谨慎为好。

另，受凶星影响，今年属鸡人容易发生意外事件，如家中被窃、钱财丢失、血光之灾等，所以属鸡人在今年要额外小心一些意外之灾，家中做好防盗工作，出门在外也要多留个心眼，保管好贵重物品。出行时要注意观察路况，遵守交通规则。

除此之外，受凶星的影响今年偏财运不佳，破财的风险比较大，所以在投资方面要多谨慎。

此外今年部分朋友情绪波动较大，时常会出现郁郁寡欢的样子。甚至部分朋友在今年会变得脾气暴躁，容易跟身边的亲人朋友或伴侣发生口角，建议与人交往时，要多注意控制自己脾气，以免因为自己的脾气而与身边的人产生隔阂。

今年“文昌”星入宫，利文化和考试运。所以今年属鸡的学生朋友学业运很不错，只要多加努力，会考得不错的成绩。
