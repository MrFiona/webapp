2020年运势吉运
2020年庚子年，岁干庚金“偏印”星坐于太岁子水劫财之上，为偏印生了“劫财”星，这个组合不为好，但太岁带了神煞“太阳”。则2020年的属猪人在事业上多会面临一些机遇，有时也为兄弟朋友所带来的机遇，但利益上需要分享，不是凭空得来，并且也需要自己多去争取。其次，在2020年属猪人男生相对自信或是好强，有一颗不屈不挠的心，而女生的话则相对消极一些。

2020年运势凶运
2020庚子年太岁子水为生肖猪“羊刃”凶星，同时太岁带了“亡神”和“咸池桃花”，并且太岁为“劫财”星，以上都为凶煞而并非吉星。这个太岁所带的“羊刃”是破财，破坏福气，和领导上司关系不对付，或是产生反抗心理。“亡神”则代表了容易引发灾祸，而劫财则是矛盾是非破财，受兄弟朋友所拖累而导致。“咸池桃花”是一颗中性的星，如果桃花运差者，则2020年多容易获得感情的突破，甚至婚姻，只是花费增多。而日常桃花运不差的，甚至有感情的属猪人在2020年则容易遭遇烂桃花，露水情缘，并且花钱增多，处理不好会有酒色之灾。

2020年财运运势
财运方面为不吉，2020年岁干为庚金“偏印”星自坐太岁子水“劫财”，此组合为劫财化偏印，是不利于竞争，容易被兄弟朋友把机遇给抢夺了，或是私下占了你的便宜。况且太岁本位“劫财”星，那就是抢劫你钱财，令你散财，破财的星，所以2020年那就得注意，远离不靠谱的兄弟朋友，远离酒色，尽量不要从事金融项目和水利项目，否则败财的可能性非常大。注意，2020的生肖猪求财需要掌握一定的尺度，不宜太过贪婪，尽量不要得罪兄弟朋友。属猪人不妨在家中或办公室摆放面向自己的【祥安阁八方运财】风水摆件来进行增旺财运，避免因为太岁带了凶煞而带来的是非破财；同时佩戴【祥安阁八方运财】坠链使用效果更佳。

2020年事业官运
生肖猪事业官运方面，因生肖猪“偏印”星庚金透出为2020年的岁干，但生了坐下太岁子水“劫财”星。那说明2020年事业上会有一些机遇，但容易被抢夺，自身竞争不利，容易遭遇小人，容易受到强势的干扰，可能就是个花架子，自己看到了，根本难以很好的去把握，陪别人走个过场。但有时也会因为兄弟朋友而带来一些机遇，但要懂得利益上的分享，否则也容易失去。不妨摆放一件【祥安阁凤仪荣归】摆件来减少小人所带来的烦恼，避免不利于竞争，学习的学业有成，工作的如鱼得水，贵人助你事业蒸蒸日上，名利双收。

2020年感情运势
生肖猪的人今年逢太岁为“咸池桃花”岁干庚金偏印透出，属猪人在2020年桃花运不错，容易获得感情上的机遇，容易遭遇快速的感情，露水情缘，容易遭遇因为欲望而彼此走到一起的感情，甚至可能会有不伦之恋，但也有酒色破财或是为了感情而散财的信号，并且有时还可能会因为争风吃醋而引发矛盾是非。无论是单身还是有伴侣的属猪男女，2020年可随身佩戴或者在床头放一条【祥安阁属猪转运手链】，本命佛守护、五行互相联系转运、六合好比夫妻之亲乃感情婚姻的吉祥饰品，寓意今年感情和合顺遂；单身人士寓意提升自信，增添魅力，期盼早日遇到良缘；而有对象的属猪人则寓意感情的和睦稳固，与伴侣恩爱如初，且避免第三者破坏感情。

2020年健康平安运势
2020生肖猪健康方面因为遭遇太岁子水为“羊刃”，并带了“亡神”，是灾祸，也是矛盾、肢体冲突，或是暴露出自己强势的一面，有时也会为兄弟朋友所拖累而强势了一把，结果可能会引来伤灾。并且太岁子水过旺，则命运中的五行金和火容易导致受伤严重，则要提防呼吸系统出现疾病或是隐患，心脏压力大等问题。不仿佩戴【祥安阁勾财添福】吊坠来消除太岁凶煞所带来的危害和影响，令自身理智得到壮大，个人约束力得到增强，从而避开太岁凶煞的隐患。 对于开车的属猪人来说，2020年不宜斗气驾驶，不宜酒后驾驶，不宜经常去江湖河边或是向北的方向，否则容易引发交通事故或是带来其它伤害。不妨在车上悬挂【祥安阁财来运转】车挂作为爱车的吉祥物，避免外出撞煞，化去随行路上的凶煞，为自己迎取吉祥的瑞气。
