1990年4月，刘嘉玲被四名大汉掳走3小时，因此这几年来各大媒体，对她失踪的这段时间，更是猜测纷纷，对其演艺事业影响非常不好。刘嘉玲是虔诚的佛教徒，每次遭受媒体打击都会到寺院潜心拜佛，至今仍在家里供奉其生肖蛇本命佛普贤菩萨，祈求保佑其平安顺利。

2010年刘嘉玲凭借《狄仁杰之通天帝国》中饰演“武则天”，荣获金像奖最佳女主角奖后演艺事业更是到达顶峰。之后刘嘉玲转向商界，在上海经营的酒吧、品牌代理等也让其赚得盆满钵满。

周润发自幼父母便给他佩戴羊之本命佛大日如来菩萨，大日如来菩萨能成就一切事业，使贵人来助，鸿图大展。周润发便是如此。毫无表演经验的周润发在应征无线电视演员训练班时险遭评委放弃，幸得考官之一的钟景辉赏识，以一票“通过”说服了其他人录取了周润发。1974年，周润发顺利从训练班毕业，经过短暂的龙套生涯，周润发就得到了演出的机会。之后才华渐露，在TVB脱颖而出，红遍大江南北。

张卫健出了自己的第一张个人专辑《真真假假》，就拿了乐坛新人奖。但是由于他接戏太多，曝光率太频繁，而且接拍的是无聊的搞笑片，所出的唱片不讨歌迷的欢心，他的声势渐渐滑落。没有人再找他拍戏、出唱片。张卫健自从佩戴了生肖蛇本命佛普贤菩萨之后一扫之前的阴霾，不久台湾现代派唱片公司邀他过档发展，接连在台湾推出的三张国语唱片，平均销量达15万张，还入选“十大偶像”。后来香港TVB电视剧《西游记》更是张卫健事业推向第三个高峰。

张学友推出的唱片《昨夜梦魂中》时销量只有几千张，歌曲流行榜成绩不尽人意，而在当年香港的几大音乐颁奖礼上，张学友亦一无所获。

后来张学友经朋友推荐，佩戴了生肖牛本命佛虚空藏菩萨后时来运转，谭咏麟和张国荣先后退出香港音乐颁奖礼，张学友抓住机会成为宝丽金唱片公司主打歌手之一。此后歌神张学友便一直站在流行音乐的顶端。

毕业后，在深圳的地铁上班。每天过着单调的生活，工资也是非常固定。后来朋友送了我一个生肖本命佛，有增进财运的效果。当时是抱着不用白不用的心态，按着朋友说的方式进行佩戴，忘记过了多长时间，我的财运真的有所增长。一个朋友介 绍我做一种化妆品的微商，偶尔在朋友圈转发，有很多小伙伴购买。经济大幅度增长。我也开始慢慢掌握了赚外快的技巧，现在收入日益渐长，再也不担心钱不够花的问题了。

王先生2018年奉请祥安阁吉祥物，赶走厄运
